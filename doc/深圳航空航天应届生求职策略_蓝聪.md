## 二、简历和面试核心能力突出策略

### 简历应突出的核心能力

#### 1. 结构设计能力（最重要）
- **CATIA 三维建模**：强调熟练程度，最好附上设计案例截图（如整机结构、关键零部件）
- **碳纤维复合材料铺层设计**：这是**稀缺技能**，大部分机械专业应届生不会，务必重点突出
  - 说明铺层角度、层数设计逻辑
  - 如果有实际减重数据（如"减重30%"），一定要写
- **AutoCAD 二维制图**：工程图绘制能力

#### 2. 无人机整机经验
- 参与过**完整无人机从设计到试飞**的流程
- 强调**飞行测试经验**：数据采集、问题排查、迭代优化
- 如果有**自研无人机**项目，务必在简历中放项目链接/视频

#### 3. 竞赛成果量化
```
不要写：参加XX无人机设计大赛
要写：中国大学生飞行器设计大赛 全国X等奖
     - 独立完成XX无人机结构设计，使用CATIA完成全三维建模
     - 采用碳纤维复合材料铺层方案，整机减重XX%
     - 通过X次迭代优化，实现续航XX分钟，载重XX kg
     - 团队X人，负责结构设计与飞行测试
```

#### 4. 工程素养
- GD&T（几何尺寸与公差）基础
- 材料力学、有限元分析基础
- 对无人机气动布局的理解
- 对制造工艺（CNC、3D打印、复材成型）的了解

### 面试高频问题准备

1. **"介绍一下你最满意的结构设计项目"** → 准备3分钟结构化回答（背景-设计-验证-结果）
2. **"碳纤维铺层设计时如何确定铺层角度和层数？"** → 从力学分析、试验验证角度回答
3. **"结构设计中出现过什么问题？怎么解决的？"** → 准备一个真实案例
4. **"你对无人机行业未来趋势怎么看？"** → 低空经济、eVTOL、AI+无人机等方向
5. **"为什么选择大疆/我们公司？"** → 对公司产品线的了解 + 个人职业规划的匹配

### ⚠️ 蓝聪当前处境分析

蓝聪2026年6月毕业，现在已经错过了2025年秋招的黄金期。但**仍有多个机会窗口**：

1. **春招（2026年2-4月）**：目前正在进行中，很多企业有补招需求
2. **秋招补录/春招补招**：部分企业秋招没招满，春招会继续开放
3. **2027届提前批（捡漏）**：部分企业对2026届未就业学生仍开放
4. **社招初级岗位**：一些中小企业对"应届/1年以内经验"岗位常年开放

## 六、额外建议

### 低空经济风口
深圳正在大力发展低空经济，2024年出台《深圳经济特区低空经济产业促进条例》，无人机/飞行汽车企业大量涌现，这是蓝聪的**时代机遇**。

### 可拓展方向
1. **eVTOL（电动垂直起降飞行器）**：峰飞、小鹏汇天等企业在深圳/广州有研发中心
2. **无人机巡检/测绘**：电力巡检、农业植保方向需求旺盛
3. **航天配套企业**：深圳有众多为航天系统提供零部件的企业

### 避免的坑
- 不要只盯大疆一家，中小企业同样有发展空间
- 不要等"完美简历"再投递，先投再优化
- 不要在简历中写"精通"，用"熟练""掌握"更稳妥
- 面试时诚实回答，不会的技术直接说"目前不了解，但我可以快速学习"
